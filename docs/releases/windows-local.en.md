# Awesome OpenClaw Manager Windows Local Desktop Edition

[English](windows-local.en.md) | [中文](windows-local.md)

The Windows Local Desktop Edition is for teams that want to install the OpenClaw Manager app on Windows first, without bundling the full WSL2 deployment toolkit.

## Best for

- existing OpenClaw environments that only need a GUI manager
- local demos and UI reviews
- lightweight internal distribution

## Included

- `openclaw-manager.exe`
- `WebView2Loader.dll`
- `README.md`
- `manager-ui.png`
- `manager-ui-en.png`

## Install

1. Download `Awesome-OpenClaw-Manager-Windows-Local-v0.0.7.zip`.
2. Extract it on Windows.
3. Run `openclaw-manager.exe`.

## Notes

- The most complete automation story in this repository is still the Windows + WSL2 path.
- If you want automatic gateway creation, service wiring, manifest updates, and bridge management, use the WSL2 edition.
