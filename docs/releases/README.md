# Awesome OpenClaw Manager 发布版本说明

Awesome OpenClaw Manager 当前整理为两个下载版本，分别服务两类不同的 OpenClaw 部署需求。

## 1. Windows 本地桌面版

适合场景：

- 你已经有可用的 OpenClaw 运行环境
- 你只想先在 Windows 本机安装管理器程序
- 你希望交付包尽量精简

查看说明：

- [Windows 本地桌面版说明](windows-local.md)
- [Windows 本地桌面版下载](https://github.com/tbszz/awesome-openclaw-manager/releases/tag/v0.0.7-windows-local)

## 2. Windows + WSL2 完整部署版

适合场景：

- 你要在 Windows 和 Ubuntu 子系统中管理多 gateway
- 你需要 WSL2 service、端口桥接和 manifest 驱动的扩容能力
- 你希望一套包里同时包含桌面程序和部署脚本

查看说明：

- [Windows + WSL2 完整部署版说明](windows-wsl2.md)
- [Windows + WSL2 完整部署版下载](https://github.com/tbszz/awesome-openclaw-manager/releases/tag/v0.0.7-windows-wsl2)
