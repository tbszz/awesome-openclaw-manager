# Awesome OpenClaw Manager Release Guide

[English](README.en.md) | [中文](README.md)

Awesome OpenClaw Manager currently ships in two downloadable editions for two different deployment paths.

## 1. Windows Local Desktop

Best for:

- users who already have a working OpenClaw runtime
- users who only want the Windows desktop app first
- smaller internal demos or lightweight installs

Read:

- [Windows Local Desktop Guide](windows-local.en.md)

## 2. Windows + WSL2 Full Deployment

Best for:

- users who run multiple OpenClaw gateways inside Ubuntu on WSL2
- users who need manifest-driven expansion, service control, and port bridging
- users who want the full desktop plus deployment toolkit

Read:

- [Windows + WSL2 Guide](windows-wsl2.en.md)
